# atlas-data-platform

## Overview
A multi-domain platform for data ingestion, transformation, and analytics.

## Stack
- Airbyte Cloud (ingestion)
- dbt Cloud (transformation)
- BigQuery + GCS (storage)
- Looker Studio (dashboard)
- Terraform (infra-as-code)
- GitHub Actions (CI/CD)

## Quickstart
1. Clone the repo
2. See docs/RUNBOOK.md for setup and run instructions
3. Infrastructure: infra/terraform
4. Transformation: dbt/
5. Ingestion: airbyte/
6. Dashboards: looker_studio/
